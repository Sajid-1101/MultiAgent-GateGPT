About GateGPT AI

Your intelligent AI powered assistant for GATE Computer Science preparation.

🤖 What is GateGPT AI?

GateGPT AI is an intelligent Agentic RAG chatbot built specifically for GATE Computer Science preparation. It combines specialized AI agents, metadata-aware retrieval, and a curated knowledge base of books, notes, solved papers, and previous year questions to deliver accurate, context-aware, and reliable responses.

⚙️ How It Works

User Query → Supervisor Agent → Concept Agent / PYQ Agent → Metadata Filtered FAISS Retrieval → Gemini AI → Verification Agent → Final Response

GateGPT AI uses an Agentic Retrieval-Augmented Generation (Agentic RAG) architecture. The Supervisor Agent first understands the user's intent and routes the query to the appropriate specialized agent. The Concept Agent retrieves information from books, notes, and solved papers, while the PYQ Agent accesses previous year questions using metadata-filtered FAISS retrieval. Finally, the Verification Agent validates the generated response against the retrieved knowledge before presenting the final answer, reducing hallucinations and improving reliability.
